<summary><strong>VedaVerse — RAG-Based Academic AI Assistant</strong></summary>

A Django-powered AI learning platform enabling intelligent interaction with academic PDFs through Retrieval-Augmented Generation. Combines semantic search, LLM reasoning, vision capabilities, and speech synthesis into a single cohesive study tool.

| Stack | Scale | Performance | Security | Impact | Repository |
| --- | ---: | --- | --- | --- | --- |
| Python, Django, LangChain, OpenAI, ChromaDB, Gemini, gTTS | Local / cloud-ready | Optimised chunking & semantic retrieval | Env-based credentials, sanitised uploads | Academic Q&A, quiz gen, audiobook conversion | [VedaVerse](https://github.com/moni04sh/VedaVerse) |

- Built a semantic retrieval pipeline using LangChain, OpenAI Embeddings, and ChromaDB for context-aware question answering over academic PDFs.
- Implemented AI-powered quiz generation using LLMs for active recall and automated self-assessment.
- Developed PDF-to-audiobook conversion using LLMs and gTTS for accessibility-first learning.
- Integrated Google Gemini Vision to extract and solve handwritten mathematical and scientific equations through a custom sketchbook interface.

</details>

<details>
<summary><strong>MediLearn — AR-Based Medical Training Platform with Explainable AI Diagnostics</strong></summary>

A full-stack medical education platform combining WebAR-based 3D anatomy visualisation with a hybrid AI diagnostic model — published in IEEE Xplore at the 2026 IITCEE International Conference.

| Stack | Scale | Performance | Security | Impact | Repository |
| --- | ---: | --- | --- | --- | --- |
| React + TypeScript, WebXR, Three.js, FastAPI, PyTorch, PostgreSQL, Docker | Multi-user platform | 95.99% accuracy, 95.98% F1-score | Auth-gated, Dockerised, input-validated | IEEE published; immersive med-ed | [MediLearn](https://github.com/moni04sh/Medilearn) |

- Built a hybrid medical image classification model using EfficientNet-B0 and Vision Transformer (ViT), achieving 95.99% accuracy and 95.98% F1-score on diagnostic tasks.
- Implemented Explainable AI using Grad-CAM heatmaps to provide visual interpretation of model predictions and improve clinician trust in AI-assisted diagnostics.
- Developed WebXR and Three.js powered immersive 3D anatomical learning experiences with real-time AI-generated insight overlays.
- Architected scalable frontend and backend systems using React TypeScript, FastAPI, PostgreSQL, and Docker for reliable deployment and user management.
- Research published in the 2026 IEEE IITCEE International Conference, indexed on IEEE Xplore.

</details>
